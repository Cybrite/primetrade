# PrimeTrade Backend API

RESTful API with JWT authentication and role-based access control.

## Quick Start

1. Install dependencies:

```bash
npm install
```

2. Configure environment variables in `.env`

3. Start the server:

```bash
npm run dev
```

## API Documentation

Visit `http://localhost:5000/api-docs` for complete API documentation.

## Features

- JWT Authentication
- Role-Based Access (User/Admin)
- Task Management CRUD
- Input Validation
- Error Handling
- Rate Limiting
- Security Headers

## Tech Stack

- Node.js
- Express.js
- MongoDB + Mongoose
- JWT
- bcrypt
- Swagger
